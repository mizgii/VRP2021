About our project


Our project means to visualise results of experiments with simulations of Pickup and Delivery models with capacity constraints and in the future, time windows constraints. At the moment by going to the simulation page, you can there enter your desired number of points (an even number) in the 'virtual city', as well as number of vehicles and the capacity of vehicles (we assume all vehicles are of the same type). Afterwards we generate a random distance matrix for the specified number of points, random indexes for pickups and deliveries and randomize values of demands according to maximum capacity of vehicles. Then as a result you get optimized routes for each vehicle and a simple visualisation of routes in a graph. Please keep in mind that the optimization might take a few seconds up to a few minutes, depending on your number of points. We request you don't go beyond 50 points for ease of analyzing the routes. Please click on the 'Simulation' link up above in order to proceed.

Tools


The tools we use for optimization are solving algorithms provided by the Google OR-tools set. The application is written in Python. The website application uses Flask framework and for now the simple Bootstrap CSS template.
